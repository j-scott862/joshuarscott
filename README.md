# joshuarscott
Repo for personal site
Static HTML/CSS portfolio site designed to be hosted on GitHub Pages.

## Local preview
Just open `index.html` in a browser.

## GitHub Pages
1. Create a repo named: `yourusername.github.io`
2. Upload the site files (root should contain `index.html`)
3. In GitHub: Settings → Pages → Build and deployment:
   - Source: Deploy from a branch
   - Branch: main / root
4. Your site will be at: https://yourusername.github.io

## PDFs
Place:
- `assets/pdf/portfolio.pdf`
- `assets/pdf/resume.pdf`

## Images
Add project images under `assets/images/` and update the filenames in HTML.
